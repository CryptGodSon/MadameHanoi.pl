# Madame Hanoi 🪷

Strona internetowa restauracji wietnamskiej **Madame Hanoi** — ul. Chmielna 6, 00-020 Warszawa.

- 🌐 Dwujęzyczna: **PL** (domyślnie) / **EN** — przełącznik w prawym górnym rogu, wybór zapamiętywany
- 📖 Pełne menu (32 pozycje) przepisane z oryginalnych kart + podgląd oryginalnych kart w lightboxie
- 🖼️ Galeria potraw, animowany hero z "lampionami", mapa dojazdu
- ⚡ Czysty HTML + CSS + JS, zero frameworków i zależności

## Struktura

```
index.html      — treść strony (PL + tłumaczenia w atrybutach data-en)
css/style.css   — style
js/main.js      — animacje, lightbox, przełącznik języka
img/            — zdjęcia potraw i karty menu
```

## Uruchomienie

Otwórz `index.html` w przeglądarce — nic więcej nie trzeba.
